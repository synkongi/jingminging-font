# jingminging-font
# 징밍잉체

징밍잉체는 에스파 카리나의 버블 손글씨를 참고하여
비공식 팬 창작 목적으로 제작된 한글 폰트입니다.

본 폰트는 해당 아티스트 및 소속사와 직접적인 관련이 없으며,
개인 감상 및 비상업적 사용만 허용됩니다.

## 포함 파일
- 징밍잉체 Medium

## 사용 범위
- 개인 사용 가능
- 비상업적 사용 가능
- 상업적 사용 및 수정·재배포 절대 금지
